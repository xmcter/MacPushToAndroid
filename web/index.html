Telegram Bot

通过 Telegram 机器人接收通知

如何获取 Bot Token 和 Chat ID？ (点击展开)
  1. 打开 Telegram，搜索并关注官方机器人 @BotFather。
  2. 给它发送 /newbot 指令，按照提示为您的机器人命名，即可获得 Bot Token。
  3. 在 Telegram 中搜索并关注您自己刚刚创建的那个机器人，并点击 Start。
  4. 然后，在 Telegram 中搜索关注机器人 @userinfobot，给它发送任意消息即可查看到您自己的 Chat ID。

电子邮件 (SMTP)

通过您的邮箱接收推送通知

如何配置邮箱 SMTP 服务？ (点击展开)
以主流的 QQ 邮箱为例：
  1. SMTP 服务器 填写 smtp.qq.com，端口 填写 465。
  2. 发件人邮箱 填写您的完整 QQ 邮箱地址。
  3. SMTP 密码/授权码：填写获取到的 16 位授权码。
  4. 收件人邮箱 填写您用来接收通知的任意邮箱。
